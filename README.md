Overview
- Minimal Node.js project with an entry point at `index.js`.
- Creates a simple server for signaling 

Prerequisites
- Node.js
- npm

Installation
- Clone the repository:
    ```
    git clone <repo-url>
    cd <repo-directory>
    ```
- Install dependencies:
    ```
    npm install
    ```

Usage
- Run the entry point directly:
    ```
    node index.js
    ```
- Or run an https server (needs certificates -> see mkcert):
    ```
    USE_HTTPS=1 node index.js
    ```
